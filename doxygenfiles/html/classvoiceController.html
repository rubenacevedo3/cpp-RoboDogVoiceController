bool voiceController::matchCommand ( const std::string &  signal)

match the command function

This function checks to see if any of the command's audioString in the commandList is in the inputed string. If it is then it outputs the command name to the motionModule and returns true. If there is no match then it returns false and does nothing. This is how the Voice Controller checks to see if the user said a trained command to the RoboDog.
